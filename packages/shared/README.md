# @hybris-mcp/shared

Internal package — exports validators and config types used by `@hybris-mcp/runtime`, `@hybris-mcp/solr`, and `@hybris-mcp/knowledge`.

```ts
import {
  validateString,
  validateNumber,
  validateBoolean,
  validateStringArray,
} from '@hybris-mcp/shared';
```

Not published independently.
