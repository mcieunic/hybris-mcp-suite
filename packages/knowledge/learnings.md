# Hybris learnings

Organicznie rosnąca baza nietrywialnych ustaleń: problem → root cause → fix → extension/area, którego dotyczy.

Format wpisu:
```
## YYYY-MM-DD — krótki tytuł
- **Problem:** opis
- **Root cause:** dlaczego (nie tylko "co naprawiło")
- **Fix:** co zrobiliśmy (z file paths gdzie sensowne)
- **Dotyczy:** extension(y), obszar (cart/cms/checkout/...)
- **Source:** PR/ticket/sesja
```

Wpisy nowsze na górze.
